# 2026-Last-Meadow-Online-Bot

这是一个专为 Discord 2026 年愚人节活动游戏《Last Meadow Online》编写的全自动挂机脚本。

## 📖 游戏背景与机制

2026 年愚人节，Discord 推出了一款名为《Last Meadow Online》的全球合作 MMO 网页挂机游戏。全服玩家被分为不同职业（如牧师、骑士等），需要共同收集物资、打怪升级，最终迎击世界 Boss 绿龙（Grass Toucher）。

游戏包含三个核心玩法（对应不同的冷却周期）：
* **Adventure（冒险）**：核心的资源挂机获取渠道。
* **Craft（合成）**：通过方向键（QTE）序列组合物资。
* **Battle（战斗）**：经典的 3x3 翻牌三消小游戏，提供巨额经验。

由于游戏重度依赖高频点击与重复操作，本脚本旨在释放玩家的双手，实现真正的全自动化打工。

## ✨ 核心特性

* **全模式制霸**：原生支持 Adventure、Craft、Battle 三大模块的自动流转。
* **智能雷达导航**：自动检测各模块冷却时间（Cooldown），技能就绪后自动切页，完成后秒回城挂机。
* **底层事件注入**：绕过 Discord React 框架的合成事件拦截（Synthetic Events），直接调用原生绑定的 `onClick` 事件，确保点击绝对有效。
* **三消最强大脑**：Battle 模块引入基于 SVG `path` 的特征值（DNA）识别算法，配合“状态隔离区”机制，避免动画延迟导致的死循环逻辑卡顿。
* **静默运行**：精简的 Console 战报输出，不会刷屏。

## 🚀 使用指南

1.  在 Discord 中打开 《Last Meadow Online》 游戏界面。
2.  按下 `F12` 或 `Ctrl+Shift+I` 打开浏览器的**开发者工具（Developer Tools）**。
3.  切换到 **Console（控制台）** 标签页。
4.  ⚠️ **关键步骤**：在 Console 面板顶部的上下文下拉菜单中（默认显示为 `top`），将其切换为对应的 iframe 环境（通常名为 `discord_activities` 或类似字样）。
5.  将 `bot.js` 中的代码复制并粘贴到控制台中，按下回车即可运行。

### 控制命令

可以在控制台中随时输入以下命令控制脚本：
* `stopBot()`：强制停止脚本运行。
* `startBot()`：重新启动脚本。
